Method MakeCommand
Creates a command object based on the provided commandType. This method checks the input command type, validates it, and returns an instance of the corresponding command. If the command type is invalid or unrecognized, an exception is thrown.
Syntax
C#
    public override ICommand MakeCommand(string commandType)
Parameters
commandType
The type of the command to create (e.g., "tri", "clear", "write").
Return Value
An instance of a command that implements the ICommand interface.
Exceptions
Exception Condition
ArgumentNullException Thrown when commandType is null or empty.
ArgumentException Thrown when the commandType is invalid or unrecognized.
Exception Thrown if there is any other error while creating the command.
